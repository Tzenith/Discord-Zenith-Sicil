###### Doldurulması Gereken Yerler

- ./commands Klasöründeki Gerekli Yerleri Doldurmanız yeterli olacaktır.
- ./ayarlar.json dosyasının içini eksiksiz doldurmanız yeterlidir.
- ./başlat.bat Dosyasına çift tıklayıp herhangibir tuşa basarak botu aktif etmeniz gerekmektedir.
- ./Son Olarak Node Modüller Yüklü Değildir Botu Başlattığınız'da Başlamayacaktır. Onun İçin Yapmanız Gerekenler
- ./Botunuzun Klasörüne Boş Bir Yerine **//Shift Tuşu + Sağ Tıklayıp\\** PowerShell Seçeneğini Seçip Oraya Şunları Yazın **// npm install \\** Ve Kurulmasını Bekleyin. Botunuz Hazır.!
